# CodeAlpha Fitness Tracker Dashboard

## Description
This Fitness Tracker Dashboard is developed using React JS as part of the CodeAlpha Internship Program.  
It allows users to log workouts, track calories burned, count steps, and monitor daily fitness progress.

---

## Features
- Add workout manually
- Track duration, calories, and steps
- Daily summary dashboard
- Progress bar for calorie goal
- Delete workout option
- Data stored using LocalStorage
- Clean and user-friendly UI

---

## Technologies Used
- React JS
- JavaScript
- HTML
- CSS
- LocalStorage

---

## How to Run

1. Install dependencies:
npm install

2. Start the app:
npm start

App runs at:
http://localhost:3000
