Description

This Flashcard Quiz Application is developed using React JS as part of the CodeAlpha Internship Program.
It allows users to answer multiple-choice questions with a timer and view their final score with performance feedback.

Features

15 Multiple Choice Questions (MCQs)

-> Timer for each question

-> Score calculation system

-> Star rating based on performance

-> Progress bar animation

-> Top 5 scores stored using LocalStorage

-> Responsive and attractive UI

-> Color indication for correct and incorrect answers

Technologies Used

-> React JS

-> JavaScript

-> HTML

-> CSS

-> LocalStorage

How to Run

Install dependencies:
npm install

Start the app:
npm start
Local Development:
http://localhost:3000
